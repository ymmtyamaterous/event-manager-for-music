1. フッターが中途半端な位置に表示されているのでwindowの最下部に固定したい

3. デバッグ用のテストデータを設定するmigration機能が欲しい
4. パスワード入力欄に目のアイコンを用意して表示/非表示が切り替えられるようにしたい
5. ログアウトボタン押下時に確認モーダルを表示してからログアウトするようにしたい
6. 現在ブラウザの標準モーダルで入力を要求している部分について、UIを実装してほしい
- 申請時のメッセージ、承認時の開始時刻、順番設定等
7. タイムテーブル管理画面で一覧をドラッグアンドドロップで並び替えられるようにしたい
8. イベントに対してフライヤー画像を設定できるようにしたい
- ファイルをアップロードして設定
9. 出演者による登録済みバンドの編集機能
- バンドのユーザーやセットリスト、バンドのプロフィール画像も設定できるようにしたい

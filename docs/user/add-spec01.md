# 見つかったバグ、追加仕様、懸念事項

## バグ
1. 運営者でログイン中に新規イベント作成時にユーザーが存在しませんというエラーでイベント作成できない
- 開発者ツールには404エラーが表示される
2. ログアウト機能がない

## 追加仕様
1. レスポンシブ対応
- 特にヘッダー部分
2. トップ画面には開催予定イベントにいくつか候補があるが、イベント画面にはイベントがないと表示される
- モックデータがトップ画面のみに表示されていると思うがモックデータは削除してほしい
3. ログイン中はログインボタンをログアウトボタンにしてほしい

## 懸念事項
1. ヘッダーのマイページを押したときにトップ画面が表示されるがこれは正しい挙動かどうか
2. 運営者ログイン中にヘッダーに出演者ボタンが表示されているが、これは出演者ユーザー用のものではないか

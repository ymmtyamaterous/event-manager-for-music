## 6. URL一覧

### 公開ページ（認証不要）

| URL | ページ名 |
|-----|---------|
| `/` | トップページ |
| `/login` | ログインページ |
| `/register` | 新規登録ページ |
| `/events` | イベント一覧 |
| `/events/[id]` | イベント詳細 |
| `/bands/[id]` | バンド詳細（公開） |

### 観客ページ（audience）

| URL | ページ名 |
|-----|---------|
| `/audience` | 観客マイページ（予約一覧） |
| `/profile` | プロフィール |

### 運営者ページ（organizer）

| URL | ページ名 |
|-----|---------|
| `/organizer` | 運営ダッシュボード |
| `/organizer/events/new` | 新規イベント作成 |
| `/organizer/events/[id]` | イベント詳細（管理用） |
| `/organizer/events/[id]/edit` | イベント編集 |
| `/organizer/events/[id]/entries` | エントリー申請管理 |
| `/organizer/events/[id]/announcements` | お知らせ管理 |
| `/organizer/events/[id]/reservations` | 予約者一覧 |
| `/organizer/events/[id]/performers` | 出演者管理 |
| `/profile` | プロフィール |

### 出演者ページ（performer）

| URL | ページ名 |
|-----|---------|
| `/performer` | 出演者ダッシュボード |
| `/performer/bands/new` | バンド登録 |
| `/performer/bands/[id]` | バンド編集 |
| `/performer/events` | イベントエントリー一覧 |
| `/performer/events/[id]/entry` | エントリー申請 |
| `/profile` | プロフィール |

---

## 7. リダイレクト仕様

| 状況 | リダイレクト先 |
|------|--------------|
| 未ログインで保護ページにアクセス | `/login` |
| JWTの有効期限切れ | `/login` |
| 権限外のロールでダッシュボードにアクセス | `/`（トップページ） |
| 存在しないリソースIDにアクセス | カスタム404ページ |
| ログアウト後 | `/`（トップページ） |
| 予約成功後 | `/audience` |
| イベント作成成功後 | `/organizer/events/[newId]` |
| バンド登録成功後 | `/performer` |
| エントリー申請成功後 | `/performer/events`（2秒後） |

## 8. 認証フロー（JWT）

```
[フロントエンド]                    [バックエンド]
       │                                  │
       │  POST /api/v1/auth/login         │
       │ ─────────────────────────────►   │
       │                                  │ bcryptでパスワード検証
       │  { access_token, refresh_token } │
       │ ◄─────────────────────────────   │
       │                                  │
       │ アクセストークンを保存            │
       │                                  │
       │  GET /api/v1/events              │
       │  Authorization: Bearer {token}   │
       │ ─────────────────────────────►   │
       │                                  │ JWTを検証
       │  200 OK                          │
       │ ◄─────────────────────────────   │
       │                                  │
       │ [アクセストークン期限切れ]         │
       │                                  │
       │  POST /api/v1/auth/refresh       │
       │ ─────────────────────────────►   │
       │                                  │
       │  { access_token }                │
       │ ◄─────────────────────────────   │
```
